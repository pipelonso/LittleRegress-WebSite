LittleRegress Rebirth install instructions

Si estas en windows oprimé la combinación de teclas Windows + R para abrir la ventana de ejecutar

If you are on windows press the Windows + R key combination to open the run window

En la ventana de ejecutar escribe "%appdata%" y pulsa ok

In the run window type "%appdata%" and press ok

Se te abrirá una carpeta que contenga otra carpeta llamada roaming, abrela.

A folder containing another folder called roaming will open, open it.

Si tienes minecraft java instalado te aparecerá una carpeta llamada .minecraft, entra en ella.

If you have minecraft java installed a folder called .minecraft will appear, enter it.

Encontraras dos carpetas:

  1. • saves
  2. • resourcepacks

You will find two folders:

  1. • saves
  2. • resourcepacks

Agrega el archivo de texturas LR2.textures.zip dentro de la carpeta resourcepacks

Add the texture file LR2.textures.zip inside the resourcepacks folder

Si entras a la carpeta saves, veras las carpetas que contienen tus mundos.

Entra al mundo al que quieres instalar el datapack.

If you enter the saves folder, you will see the folders that contain your worlds.

Enter the world where you want to install the datapack.

Dentro de la carpeta de tu mundo, abre la carpeta datapacks

Y mueve el archivo littleregressdatapack.zip dentro de ella.

Inside your world's folder, open the datapacks folder

And move the littleregressdatapack.zip file inside it.

Ya puedes abrir minecraft

You can now open minecraft

Ve a la configuración de paquetes de texturas para activar las texturas del datapack.

Go to the texture pack settings to activate the datapack textures.

Activate the resource pack
